# 🎓 EduPro Learning Platform Dashboard

## 📌 Project Overview

The **EduPro Learning Platform Dashboard** is an interactive Power BI project developed to analyze user enrollment, course performance, teacher ratings, and demographic trends within an online learning platform.

This dashboard transforms raw educational data into meaningful insights that help stakeholders understand learner behavior, course popularity, and instructor performance. It enables data-driven decision-making through dynamic visualizations and KPIs.

## 🎯 Purpose
The primary objective of this project is to:
- Analyze student enrollment patterns.
- Monitor course distribution by level and type.
- Identify the most active age groups.
- Compare enrollment trends across gender.
- Evaluate teacher performance based on ratings.
- Analyze course ratings across different categories.
- Present educational insights through an interactive Power BI dashboard.

## 🛠️ Tech Stack

- **Power BI** – Dashboard Development & Data Visualization
- **Microsoft Excel** – Data Preparation and Cleaning
- **Power Query** – Data Transformation
- **DAX (Data Analysis Expressions)** – Measures and Calculated Columns
- **Git & GitHub** – Version Control and Project Hosting
